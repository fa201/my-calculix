CGX - quick reference

Entity selection

n nodes, e elements, f faces, se sets, p points, l lines, c combined lines, s surfaces, b bodies, t node texts, S nurb surfaces, L nurb lines, sh shapes and si shaded surfaces.
Catch several entities: type a all, define corners of rectangular picking area with 2 r, to catch only the entity closest to you type i, move the pointer over the entities and press one of the above command.
↑ previous command and ↓ next command.
PAGE UP previous set (last command was plot or plus) or previous loadcase. Likewise PAGE DOWN next set or loadcase.
