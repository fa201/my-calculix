CGX - tips

General

Installation and set up

For Linux Ubuntu, copy ccx programs from the website and paste at /usr/local/bin. Then make an alias to cgx to launch directly from the folder of the study. Make sure the program has execute right (7xx with chmod)

Archives

Old versions of CGX are kept on the official website. You need to specify the type of files and the version in the URL. Here are all files related to CGX 2.13:
